EGL Details#

Embedded Graphics Library (EGL) is an interface between the Khronos rendering APIs, such as OpenGL ES, and the underlying native platform window system. It handles graphics context management, surface/buffer binding, rendering synchronization, and enables high-performance, accelerated, mixed-mode 2D and 3D rendering by using other Khronos APIs.

Jetson Linux Driver Support supports the EGL 1.5 specification, Khronos Native Platform Graphics Interface.

GL and Vulkan Details#

The OpenGL driver in this release supports OpenGL4.6 (https://www.khronos.org/registry/OpenGL/specs/gl/glspec46.core.pdf).

  • Refer to https://www.khronos.org/registry/OpenGL/index_gl.php for more information about GL/GLX and other related specifications.

  • Refer to https://www.khronos.org/conformance/adopters/conformant-products/opengl for more information about the Conformance details for X11 and the Wayland Windowing System.

The OpenGL ES driver in this release supports OpenGL ES Common Profile Specification 3.2.

  • Refer to the Khronos OpenGL ES API Registry for more information about OpenGL ES.

  • Refer to https://www.khronos.org/conformance/adopters/conformant-products/opengles for more information about the Conformance details for X11 and the Wayland Windowing System.

The Vulkan driver in this release supports VK1.3 (https://www.khronos.org/registry/vulkan/specs/1.3/pdf/vkspec.pdf).

  • Refer to https://www.khronos.org/registry/vulkan/ for more information about the specification.

  • Refer to https://www.khronos.org/conformance/adopters/conformant-products/vulkan for information about Conformance.
